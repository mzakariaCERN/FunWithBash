## sed
 ```sed 's/;/\n /g'  Place_Holder.txt > Fixed_Place_Holder.txt```
 
 This code:
 1. replaces for all string 's' 
 2. ';' with a new line '\n'
 3. using a global flag '\g' 
 4. from a file called Place_Holder.txt 
 5. pipe it to Fixed_Place_Holder.txt
